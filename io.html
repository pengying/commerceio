Checkout Integration Solutions

  • Buy Now button
  • Online store gadget

  • Customizable shopping carts

Buy Now Button

  • Sell a single item per transaction
  • Just copy & paste pre-generated HTML

Online Store Gadget

  • Multiple items per transaction
  • Manage inventory in Google Docs
  • Instantly embed on Google Sites, Blogger, iGoogle

Shopping Cart

  • Customize to match website look and feel
  • Wide range of integration options

Adding a Shopping Cart 

// Include Google shopping cart script	
	
<script id='googlecart-script' type='text/javascript' 
  src='https://checkout.google.com/seller/gsc/v2_2/cart.js?mid=MID'
  currency='USD'>
</script>

// Add CSS annotations to products

<td class='product'>
 <img class='product-image' src='cup.jpg'>
 <div class='product-title'>Travel Mug</div>
 <div class='product-price'>$8.02</div>
 <div class='googlecart-add-button'></div>
</td>

Customizing the Shopping Cart 

// Control Location

<div id="googlecart-widget-control" 
     googlecart-widget-control-align="left"
     style="text-decoration:underline; height:30px">View cart</div>

// Customize Style

.googlecart-override .googlecart-widget-price {color: green;}
.googlecart-override #googlecart-widget-body {border-top-width: 4px;}

// Customize Behavior

highlight-color="#FF7878"
cart-opening-time="1500"

Key Take-Aways

  • Monetization is more than just payments
  • Now is a great time to think about Commerce and Payments
  • Challenges remain: fraud, compliance, and conversions
  • Google Checkout offers a flexible API and tools to build a business or add to revenues
